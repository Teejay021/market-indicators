# Market Indicators

Take-home assignment: Web app that displays market indices with 30-day detail views.

## Requirements

- List view of market indicators
- 30-day detail view with chart
- Server-side API calls with caching (60-120 sec refresh)
- Rate limiting (20 req/min, 500 calls/month)
- Optional: WebSocket stream

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Tech Stack

Next.js 15 + TypeScript + TailwindCSS
